EVM Management System

This project deals with the lifecycle of a Electronic Voting Machine(EVM), which are used across the country (India) during elections for voting. This software consists of modules like Dashboard, User Management, Warehouse Management, Order Management, Unit Management, Consumables Management, Support, Reports and Admin Utilities. This software is used by election authorities and has a hierarchical role based approach for authorizing logged in users to performs tasks or use functionalities.

Jasper was used for generating reports

Key functions include:

  • Inventory Management
  • Allocation & Deployment
  • Maintenance Tracking
  • Real-time Monitoring
  • User Access & Roles
  • Integration with other Election Systems
